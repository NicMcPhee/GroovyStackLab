Contains an interface and implementation of a basic stack class. Used in lab exercises in the University of Minnesota, Morris, CSci 3601 (Software Design and Development) course.

Package Specification

  • StackIF is the interface that defines what a stack should be.
  • Stack should (eventually) become a complete implementation of that interface.
  • StackUnderflowException is a runtime exception that is thrown when a stack underflows, i.e., when we pop or top an empty stack.s
